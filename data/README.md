# Data Sources

Synthetic source-style data for asset and wealth management BI automation analytics.

- `entities.csv`: 36 finance operations workflow records.
- `daily_metrics.csv`: 5,040 finance operations workflow-day metric rows.
- `source_events.csv`: 760 operational events, QA checks, experiments, and stakeholder asks.
- `recommended_actions.csv`: 220 candidate actions with effort and expected lift.
